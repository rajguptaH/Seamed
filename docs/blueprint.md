# **App Name**: SeaMed Inventory

## Core Features:

- Ship Creation: Create new ship profiles with vessel details like flag and crew number.
- Medicine Database: Maintain a database of medicines and equipment required based on maritime flags.
- Inventory Autopopulation: Automatically populate medicine and equipment list with required quantities based on selected flag and crew number.
- Batch & Expiry Tracking: Update and track medicine batch numbers and expiry dates.
- Expiry Notifications: Generate lists of medicines expiring in 1-month, 3-month, and 6-month intervals.
- Inventory Export: Download the medicine inventory with expiry dates in Excel format.

## Style Guidelines:

- Primary color: Deep blue (#2E5266), evoking the sea and trustworthiness.
- Background color: Light gray (#E0E5E7), offering a clean and calming backdrop.
- Accent color: Muted teal (#63A375), providing a subtle contrast that complements the primary color without being distracting.
- Body and headline font: 'PT Sans' for a modern and easily readable design.
- Use simple, clear icons to represent medicines, equipment, and actions (e.g., export, update).
- Design a clean and intuitive layout for easy navigation and quick access to essential information.
- Incorporate subtle animations for user feedback (e.g., confirmation on updates) and smooth transitions.
